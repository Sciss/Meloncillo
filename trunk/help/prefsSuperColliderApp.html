Plug-Ins Prefs : SuperCollider Synth Application (scsynth)

Meloncillo does not care about audio processing at all. The value specified here is available to LISP scripts via (gethash "SUPERCOLLIDERAPP" (gethash "PREFERENCES" cillo)). Scripts using offline SuperCollider can pass this value to the (execute) command. The path specifies the synthesis server, usually something like "/usr/bin/scsynth" or "/Applications/SuperCollider_f/scsynth".
