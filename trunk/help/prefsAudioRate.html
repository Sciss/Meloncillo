Plug-Ins Prefs : Audio Rate

Meloncillo does not care about audio processing at all. The value specified here is available to LISP scripts via (gethash "AUDIORATE" (gethash "PREFERENCES" cillo)). It may be used in those scripts to be passed to the offline bounce application, for example.

Note that this field doesn't influence realtime SuperCollider which is started manually. However, for the plug-ins to not get confused, this value should reflect the current sample rate of scsynth!
