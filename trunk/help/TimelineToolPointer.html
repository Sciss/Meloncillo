Pointer Timeline Tool

The pointer tool (keyboard F1) is used to set the timeline position and selection. When using other tools, this can also be accomplished by using the timeline axis.

Clicking anywhere in the trajectory view panel will set the timeline position (the red vertical bar) which indicates the current playhead position and insertion point for paste operations. To quickly return to the session beginning, press Return when the timeline frame is focussed.

Click+Drag will select a portion of the timeline for cut/copy/delete operations (see Edit menu) or trajectory filtering. This selection is also used in loop playback. To extent the current selection, use Shift+Click.

Meta+Click will select the whole session span (same as Edit->Select All). Ctrl+Drag will drag the timeline position without making any selection. Double-Click will start playback.

A 'bird's view' of the timeline position and selection is shown in the timeline scrollbar.
