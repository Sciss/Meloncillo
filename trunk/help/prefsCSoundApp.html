Plug-Ins Prefs : CSound Application

Meloncillo does not care about audio processing at all. The value specified here is available to LISP scripts via (gethash "CSOUNDAPP" (gethash "PREFERENCES" cillo)). Scripts using CSound can pass this value to the (execute) command. The path specifies the command-line version of CSound, usually something like "/usr/bin/csound" or "/usr/local/bin/csound".
