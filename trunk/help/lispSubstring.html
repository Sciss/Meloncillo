LISP Functional Extensions : substring

(substring <str> <start> <stop>)

returns the portion of the passed string str starting at character index start (inclusive) and stopping at character index stop (exclusive). character indices are counted from zero.

example:

    (let ((tough-guy "donald rumsfeld"))
        (println (concatenate 'STRING
            (substring tough-guy 0 3)
            (substring tough-guy 6 10)
            "!"
        ))
    )
