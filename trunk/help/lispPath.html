LISP Functional Extensions : Pathname commands

(path-concat <parent> <child>)
(path-split <pathname>)

path-concat creates a new string describing the path to file child in directory parent using the native file separators (e.g. "/" on mac and linux, "\" on windows). path-split splits the given pathname into the file name and the directory containing that file using the native file separators, returning a cons cell whose car is the directory and whose cdr is the file.

example:

    (let ((afile (temp-file-make)))
        (let ((splitted (path-split afile)))
            (println (concatenate 'STRING "created the file '"
                (cdr splitted)
                "' in the temporary directory '"
                (car splitted)
                "'."
            ))
        )
        (file-delete afile)
    )
