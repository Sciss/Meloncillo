Filter Trajectories Dialog

The Filter Trajectories dialog offers a list of plug-ins that may transform the trajectories of the selected transmitters within the current timeline selection. The 'Plug-In' combo box lists all available plug-ins. The 'Resampling' field and the local 'Settings' menu are currently not used. The bottom half of the dialog shows the plug-in parameters. Pressing the 'Close' button will cancel the dialog. Pressing the 'Render' button will run the plug-in and replace the trajectory data when rendering is finished.

Note that in version 0.72 due to a bug, the dialog has to be closed and re-opened when you selected different transmitters!

Trajectory filtering will integrate blending if activated.

The following plug-ins are installed:

  • Time Warp
  • Vector Transformation
  • Lisp Plug-In
