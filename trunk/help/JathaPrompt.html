Jatha Diddler

The Jatha Diddler is a tiny prompt window which allows you to test LISP expressions and do brief calculations. Each expression is evaluated separately when you hit the Return key and results are displayed in the main log pane.

For for more details about the capabilities of the Java LISP interpreter Jatha by Micheal Hewett, refer to the Jatha website. For implications of using jatha inside meloncillo and for a list of extra functions, refer to the lisp overview document.
