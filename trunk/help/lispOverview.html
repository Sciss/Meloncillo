Writing LISP Plug-Ins

each of the three plug-in interfaces in meloncillo -- realtime synthesis, bouncing to disk and trajectory filtering -- offers a lisp script mode. this allows you to expand the functionality and implement new spatialization models without having to go into the actual meloncillo java source code. the lisp dialect is a very limited but compact subset of common lisp and is provided by the jatha library.

this help manual is not intended to teach any lisp. its purpose is to outline the possibilities of jatha and the way the plug-ins are accessed. the supercollider amplitude matrix plug-in is described in detail as a kind of case study.
