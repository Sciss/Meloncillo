Keyboard Shortcuts

Note: the shortcuts have been tested on Mac OS with German Apple Pro keyboard. There may be inconsistencies with other configurations or smaller keyboards (PowerBooks for example). Please report any problems and I try to fix them.

The key modifier Meta is the platform's default menu key, which is the Apple-key on Mac OS and Ctrl on Windows and Linux(?).

Global Commands

Note: windows without menu bars (this help frame for example) do not react to global commands. This will be fixed in a future version.

   Transport:

  • Space or Numpad-0 : toggle play / stop
  • Numpad-/ : toggle loop playback
  • Numpad-* : enabled / disable realtime plug-in
  • G : open go-to-time dialog

   Editing:

  • F1 ... Fn : select tool
  • Caps lock : toggle blending mode
  • V : toggle catch mode

Surface Frame

  • Delete (Backspace) : remove selected receivers
  • Meta+X/C : cut/copy selected receivers
  • Meta+V : paste receivers in clipboard to the surface
  • Meta+A : select all receivers

   Geometric offline tools:

  • Return after the initial drag gesture is complete : render trajectory tracks
  • Ctrl+Return : render trajectory tracks, move timeline selection forward and start new gesture (concatenation mode).
  • Escape : abort drag gesture

Timeline Frame

  • Return : stop playback and go to start of session
  • Shift+Return : extend selection to session start
  • Alt+Shift+Return : extend selection to session end
  • Cursor Left/Right : scroll to begin/end of selection
  • Alt+F : fit visible span to selection
  • Alt+A : zoom all the way out
  • Delete (Backspace) : remove selected time span from selected transmitters (blue shaded area)
  • Meta+X/C : cut/copy selected time span from selected transmitters (blue shaded area)
  • Meta+V : paste clipboard content to selected transmitters at current timeline position.
  • Meta+A : select whole session span

Receiver Sensitivity Editor

  • Meta+C : copy sensitivity tables to clipboard
  • Meta+V : paste clipboard content to sensitivity tables

Special Gadgets

   Path Fields:

  • Meta+Shift+Numpad-1 ... Numpad-9 : store directory in user preferences 1 ... 9
  • Meta+Numpad-1 ... Numpad-9 : recall directory from user preferences 1 ... 9
