This software comes with a few free libraries which are installed in the libraries folder in binary format. Their sourcecode is publically available on the net or -- in consonance with ¤6 b of the LGPL -- on request through the author. these libraries are:

  • Jatha by Micheal Hewitt, a java environment for a subset of Common Lisp. Jatha is used to program plug-ins for Meloncillo. License: LGPL
  • MRJAdapter by Steve Roy, which improves Mac OS look-and-feel without sacrificing platform independance. Artistic License
  • Liquid Look-And-Feel, a graphical skin used on Linux and Windows, resembling Apple's Aqua on Mac OS X. LGPL

Meloncillo makes use of the following third party sourcecodes:

  • Parametric Noise Generator by Paul Bourke.
  • BrowserLauncher by Eric Albert, used to open the system's default web browser.
