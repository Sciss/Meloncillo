Observer Palette

The observer palette displays context information about objects that you work with. Some of this information is editable. The palette consists of four socalled 'tabs' (tabulars) which will be selected automatically when you perform certain actions, but which can also be selected manually. The tabs are:

  • Cursor (Csr) is a read-only information tab showing useful information about the current mouse pointer location
  • Receiver (Rcv) shows the name and bounds of a receiver
  • Transmitter (Trns) shows the name of a transmitter
  • Group (Grp) shows the name and image path of a group
  • Timeline (Time) shows the current timeline selection
