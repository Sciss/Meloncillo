Main Log Pane

The log pane outputs different kinds of text information:

  • descriptions of potentially time consuming actions such as saving a document or pasting clipboard content, rendering trajectory data etc.
  • text output from the Jatha Diddler or LISP plug-ins.
  • runtime java errors and debugging strings (yes, this is a beta version)

The window can be cleared by selecting 'Delete' from the Edit menu when the main log is focussed.

The title of the window shows the name of the current session. A small filled circle before the session name indicates that the session has been modified since it was last saved.
