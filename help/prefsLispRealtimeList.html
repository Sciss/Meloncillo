Plug-Ins Prefs : Lisp Realtime Source List

All LISP sources shown in the 'Source' combo box of the realtime plug-in dialog's LISP plug-in are listed in a special XML file which is specified here. You can edit this XML file to add your own LISP sources or remove unused sources. The file is usually located at "$APPDIR:plug-ins/lisp/realtimelist.xml".

The XML file has a very simple format: It's root node is <synthcontrollist> and contains a list of <synthcontrol> elements, each having the attributes name (a key used for preference storage), screenname (a string displayed to the user), and file (a relative pathname specifying the LISP script file).
